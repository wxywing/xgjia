# 统一增量爬虫 — 四合一自动化

## 创建文件

### `scripts/crawl_incremental_all.py`（统一脚本）

四大类型、共享 DB/HTTP 基础设施，对比后只抓增量：

| 命令 | 数据源 | 对比字段 |
|------|--------|---------|
| `--type lofts` | gdgp.chinaxinge.com/gplist.asp（分页） | gp_id → lofts.gp_id |
| `--type races` | gdgp.chinaxinge.com race_rclist.asp?gp_id=X（逐公棚） | source_id → races.source_id |
| `--type shops` | chinaxinge.com netshop.asp?page=N（分页） | shop_id → shops.source_id |
| `--type pigeons` | 逐展厅抓展品列表（轮转策略） | product_id → pigeons.source_id |

**所有类型统一行为：**
- INSERT IGNORE 防重复
- 公棚/展厅/铭鸽自动抓详情页
- 赛事自动全量抓成绩明细
- --dry-run 只报告不写入

**铭鸽轮转优化：** `--max-shops N` 可以限制每次检查 N 个展厅，按日期 hash 确定起始偏移，保证逐步覆盖全部 366 个展厅。

### `scripts/cron_incremental_all.sh`（定时包装）

两种模式：
```
cron_incremental_all.sh          # quick: races + pigeons(30 shops)
cron_incremental_all.sh full     # full: all 4 types
```

建议 crontab：
```
0 */6 * * * ./cron_incremental_all.sh         # 每6小时 fast
0 2 * * 0   ./cron_incremental_all.sh full    # 每周日全量
```
