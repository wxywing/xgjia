## 一、现状分析

| 维度 | 现状 | 影响 |
|------|------|------|
| 域名年龄 | ~1个月 | 百度沙盒期，收录慢 |
| 文章数量 | 19篇（含本次新增3篇） | 内容太少，无收录价值 |
| 外链数量 | 0 | 权重无法传递 |
| 自然流量 | 0 | 零曝光 |
| 核心优势 | 1270万条赛绩数据 | 独家内容，可围绕数据做文章 |

**结论**: 前3个月是打基础期，重点是**持续产出内容 + 建设外链**，不能急于求成。

## 二、P0 立即执行（本周开始）

### P0-1: 长尾关键词文章 — 每周2-3篇

**目标**: 覆盖赛鸽圈低竞争、高意图搜索词，吸引精准流量。

#### 关键词挖掘方法

1. **百度下拉框**: 在百度搜索"赛鸽"、"信鸽"、"足环号"等核心词，记录下拉推荐词
2. **百度相关搜索**: 滚动到搜索结果页底部，记录"相关搜索"推荐词
3. **竞争对手**: 查看同行网站的文章标题，提取他们排名好的关键词
4. **问答平台**: 在百度知道、知乎搜索赛鸽问题，提取问题中的关键词

#### 已规划的长尾词（可立即写）

| 关键词 | 搜索意图 | 文章角度 |
|--------|---------|---------|
| 赛鸽足环号怎么看 | 信息查询 | 足环号编码规则 + 查询工具推荐 |
| 2026年春赛成绩怎么查 | 工具使用 | 查询方法 + 数据解读 |
| 鸽子血统书怎么看 | 知识学习 | 血统书阅读指南 |
| 信鸽足环号查询 | 工具使用 | 在线查询工具 + 使用演示 |
| 赛鸽分速怎么算 | 知识学习 | 分速计算公式 + 行业标准 |
| 公棚怎么选 | 决策参考 | 选公棚的5个指标 |
| 鸽子配对怎么配 | 知识学习 | 配对原则 + 血统搭配 |
| 信鸽疾病预防 | 知识学习 | 常见疾病 + 预防措施 |

#### 文章要求

- **长度**: 1500-2500字（长文更容易排名）
- **结构**: H1/H2/H3大纲清晰，配图表/表格
- **内链**: 每篇至少3个站内链接（指向工具页/数据页）
- **外链**: 引用权威来源（中鸽协、中信网等）
- **CTA**: 文末引导使用网站工具（如"立即查询足环号"）

#### 发布节奏

- 第1个月: 每周2篇（共8篇）
- 第2个月: 每周3篇（共12篇）
- 第3个月: 每周3篇（共12篇）
- **目标**: 3个月累计50篇优质文章

### P0-2: 百度系产品覆盖

#### 2.1 百度知道 — 自问自答×5

**目的**: 抢占问答流量，引导用户访问网站。

**操作方法**:
1. 注册3-5个百度账号（不同手机号）
2. 提问和回答间隔48小时（避免被判定作弊）
3. 回答中自然植入网站链接（不要硬广）

**已准备Q&A**（见 `docs/外链推广方案_20260626.md`）:
- Q1: 鸽子脚上的环号怎么查是哪里的鸽子？
- Q2: 赛鸽分速多少算快？一般多少正常？
- Q3: 新人养信鸽，怎么选公棚参加比赛？

**新增Q&A**:
- Q4: 信鸽足环号查询哪个网站准确？
- Q5: 2026年春赛成绩在哪里查？

#### 2.2 百度文库 — 上传PDF×3

**目的**: 利用百度文库的高权重，传递外链+品牌曝光。

**内容方向**:
1. 《2026年春赛数据报告》— 基于1270万条数据
2. 《赛鸽足环号编码手册》— 完整地区代码对照表
3. 《公棚选择指南》— 基于数据分析的选公棚方法

**要求**:
- PDF中嵌入网站链接（页眉/页脚）
- 标题含关键词（如"2026春赛成绩查询"）
- 上传时用百度账号（最好老账号）

#### 2.3 百度百科 — 补充参考资料

**目的**: 在现有赛鸽相关词条中，补充参考资料链接。

**目标词条**:
- 搜索"赛鸽"、"信鸽"、"公棚"等词条
- 点击"编辑"或"补充参考资料"
- 在参考资料中填入 `www.xgjia.com` 的相关页面

**注意**: 百科审核严格，不要直接填首页，要填具体内容页（如工具页、数据页）。

**目的**: 利用独家数据，产出被引用的报告，获取自然外链。

## 七、风险提示与应对

### 风险1: 百度沙盒期延长

**表现**: 3个月后仍无收录  
**应对**: 坚持产出内容，同时加大外链建设力度

### 风险2: 友链交换失败

**表现**: 邮件发出后无回复  
**应对**: 降低目标站要求，先换小站，积累到一定权重再换大站

### 风险3: 社群推广被踢

**表现**: 被群主踢出群  
**应对**: 调整话术，更自然地在帮助他人时推荐，不要硬广

### 风险4: 文章排名不佳

**表现**: 文章发布后无排名  
**应对**: 检查关键词竞争度，调整关键词策略；同时加大该文章的外链建设

## 八、关键建议（务必遵守）

1. **内容第一，外链第二**:
   - 没有优质内容，外链效果有限
   - 前2个月重点是持续产出内容

2. **数据是我们的核心竞争力**:
   - 围绕1270万条数据做内容，别人抄不了
   - 数据报告是最容易获取自然外链的内容形式

3. **耐心，耐心，耐心**:
   - SEO是慢工出细活，前3个月可能看不到明显效果
   - 但只要坚持执行，第4-6个月会开始有起色

4. **不要作弊**:
   - 不要用黑帽SEO（隐藏链接、关键词堆砌等）
   - 百度知道回答要自然，不要硬广
   - 友链交换要相关，不要换垃圾站
