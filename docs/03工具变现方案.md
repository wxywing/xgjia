## 工具先行变现方案

**编写日期**: 2026-06-22
**状态**: 方案阶段
**前提**: 网站当前零流量，会员付费不可行。改为先做鸽友刚需付费工具，工具自带流量，再用工具引导会员升级。

---

## 核心思路

不做"卖会员"，而是 **工具即流量入口**。做一个鸽友不查不行的工具，工具本身就是流量入口，用了就想付钱。

---

## 三个付费工具

### 1. 足环号深度查询报告 — ¥9.9/份

**痛点**：鸽友买鸽子、捡到鸽子、查成绩，第一个动作就是查足环号。现在全网没有一个好用的查询工具，都在各大公棚网站零散搜。

**已有基础**：`/race/ring/{number}` 已能查到跨赛季成绩（12.7M 行数据里匹配）

**产品设计**：
- 免费版：查鸽主 + 最近 3 场比赛成绩
- 付费版 ¥9.9：
  - 完整参赛履历（全部场次，含每场排名/分速/空距）
  - 分速趋势图（职业生涯速度曲线）
  - 冠军记录（如有）
  - 参赛公棚列表
  - 同鸽主其他铭鸽推荐
- PDF 证书版 ¥19.9：一键生成可打印的足环号深度报告证书

**变现逻辑**：买鸽子前查足环是刚需。花十几块钱验证一只几千块的鸽子，鸽友愿意。中信网查足环要一个个公棚翻，你这里是全量。

**自带流量**：每份报告都可以分享链接/截图 → 鸽友群传播 → 精准用户引入。

**开发量**：小。基础查询已有，只需加 PDF 生成 + 支付门控。

---

### 2. 血统证书 PDF 生成器 — ¥9.9/份 或 ¥49.9/年

**痛点**：卖鸽子必须有血统书。现在鸽友要么手写、要么 Photoshop 套模板、要么找淘宝做。淘宝做一份 30~80 元。

**产品设计**：
- 在线填写：鸽主、环号、父母鸽信息、祖父母鸽信息
- 可选模板：中式传统 / 现代简约 / 欧式古典
- 一键导出可打印的 PDF 血统证书
- 历史证书管理（年付用户可随时编辑重生成）

**变现逻辑**：替代淘宝 30~80 元的 Photoshop 服务，你 ¥9.9 自动化生成。卖方必买，不需要流量——卖鸽子的人会自己找上门。

**开发量**：中。需引入 TCPDF/FPDF 库，纯后端，半天可出 demo。现有 pedigree 模块的数据模型可直接复用。

**数据依赖**：零。用户自己填写信息，完全不依赖外部数据源。

---

### 3. 公棚对比工具 — ¥19.9/次

**痛点**：鸽友每年交鸽子前，最头疼的就是"选哪个公棚"。要对比规程、历年成绩、归巢率、参赛费、冠军含金量。现在全是手工翻网页。

**对比维度**：
- 参赛费
- 历年冠军分速
- 近 3 年平均归巢率
- 总赛事场数
- 冠军鸽数
- 最新竞赛规程（2026 赛季）

**变现逻辑**：选公棚是鸽友年度最大决策，几千上万的参赛费都花了，十几块钱验证决策太值了。

**数据依赖**：需要持续更新。数据来源于增量爬虫 + 用户补充 + 公棚合作。

**开发量**：中。lodges + races + race_results 三表联查 + 对比 UI。

## 会员体系重新定位

不卖"会员"，改为自然升级漏斗：

```
购买任意付费工具 → 自动赠送铜牌会员 30 天
     ↓
引导发布铭鸽/展厅信息
     ↓
到达发布配额上限 → 自然升级银牌/金牌
```

铜牌是获客成本，银牌/金牌才是利润来源。

## 支付接入

三个工具都需要接入真实支付：
- **微信支付**（JSAPI 或 H5 支付）
- **支付宝**（当面付或电脑网站支付）
- 个人可申请（需要个体工商户或企业资质，或者使用第三方支付聚合平台）
