# 贡献指南

感谢您关注信鸽之家项目！欢迎通过以下方式参与贡献。

## 🤝 如何参与

### 1. 反馈问题
在 [GitHub Issues](https://github.com/wxywing/xgjia/issues) 中提交：
- 数据错误报告（足环号、赛事成绩等）
- 功能建议
- Bug 反馈（请附上复现步骤）

### 2. 数据贡献
如果您有赛鸽血统数据或公棚信息愿意分享，欢迎：
- 通过微信（yingzaovip）联系我们
- 或在 Issues 中说明数据来源和范围

### 3. 代码贡献
欢迎提交 Pull Request：
1. Fork 本仓库
2. 创建特性分支 `git checkout -b feature/your-feature`
3. 提交改动 `git commit -m 'Add: your feature'`
4. 推送到分支 `git push origin feature/your-feature`
5. 在 GitHub 上提交 Pull Request

## 📋 开发说明

- 架构：原生 PHP MVC（无框架）
- 数据库：MySQL 5.7+
- CSS：集中式 `style.css`
- 所有视图使用 `_head.php` / `_footer.php` 组件
